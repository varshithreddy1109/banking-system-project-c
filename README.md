# Banking System in C

## 📌 Description
A Banking System project implemented in C using Hash Table with Linked List chaining for efficient account management. The system supports account creation, login authentication, PIN reset, deposits, withdrawals, and balance inquiry.

## 🚀 Features
- Create new bank account
- Login with account number and PIN
- PIN reset using DOB verification
- Deposit money
- Withdraw money with balance check
- View account balance
- Hash table-based account storage
- Collision handling using linked list

## 🛠️ Technologies Used
- C Programming
- Data Structures (Hash Table, Linked List)
- Dynamic Memory Allocation (malloc)

## ▶️ How to Run
1. Compile the code:
   gcc main.c
2. Run the program:
   ./a.out   (Linux/Mac)
   a.exe     (Windows)

## 📚 Learning Outcome
- Understanding of hashing and collision handling
- Practical implementation of linked lists
- Basic system design in C
